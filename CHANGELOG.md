- **zh-TW.json 完整性** - 修复了繁体中文翻译文件缺失内容的问题

## [更早版本]

### 新增
- **i18n 国际化支持** - 为整个应用添加了完整的国际化支持
  - 创建了英文 (en.json) 和简体中文 (zh-CN.json) 翻译文件
  - 所有视图组件更新为使用 `useI18n()` 和 `t()` 函数
  - 翻译内容按模块组织：login, brand, layout, menu, home, rebateSummary, rebateRecords

- **侧边栏折叠功能** - 实现了左侧侧边栏的折叠/展开功能
  - 折叠时宽度为 64px，展开时为 200px
  - 右侧内容区域会自动扩展填充可用空间
  - Logo 文字会根据折叠状态自动切换显示

- **全窗口布局** - 移除了所有页面的默认内边距
  - 更新了 style.css 以移除 body 和 #app 的默认内边距
  - 移除了 el-main 和内容容器的内边距
  - 所有页面现在占据完整窗口空间

- **返佣汇总页面** - 创建了返佣汇总页面 (/rebate-summary)
  - 包含统计卡片区域（我的返佣、自返佣、客户交易手续费等）
  - 搜索表单（UID、备注、用户类型、币种、日期范围）
  - 数据表格展示详细交易记录

- **返佣记录页面** - 创建了返佣记录页面 (/rebate-records)
  - 树形交易类型选择器（合约、现货、ETF）
  - 日期范围选择器，含"昨天"和"今天"快捷按钮
  - 数据表格展示预估返佣、实际返佣、发放状态等

- **登录页优化** - 根据设计稿重新设计了登录页面
  - 深色背景 (rgb(28, 30, 40))
  - 左右分栏布局
  - 表单输入框样式优化
  - "记住登录信息"复选框
  - 密码显示/隐藏切换

### 改进
- **MainLayout 组件** - 完善了主布局组件
  - 侧边栏导航菜单
  - 顶部工具栏（折叠按钮、返佣比例选择、语言选择、用户下拉菜单）
  - 消息横幅区域
  - 语言切换功能（10种语言支持）
  - 用户登出功能

- **路由配置** - 更新了路由配置
  - 添加了 /rebate-summary 路由
  - 添加了 /rebate-records 路由

### 技术栈
- Vue 3 + Composition API
- Element Plus UI 组件库
- Vue Router 路由管理
- Vue I18n 国际化
